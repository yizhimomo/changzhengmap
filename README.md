# 中国工农红军长征路线图

一点开，就能沿着 190+ 个历史节点重走长征路。

---

**查找你身边的长征地点** — 在地图上找到你的家乡，看看附近有哪些长征遗址；或者直接搜索地名，看看红军在这里经历过什么。

**跟着时间线走** — 拖动底部的滑块，从瑞金出发一路看到陕北会师。选中任意节点，时间轴自动跳转，地图、信息、日期同步联动。

**聚焦一支队伍** — 红一、红二、红四方面军，还是红二十五军？点一下图例，只看你想看的那条路线。

**点开一个节点** — 每一站都有详细介绍：发生了什么战斗、谁指挥、哪些部队参战、史料怎么记载，还有沙盘推演视频。

**右键看看附近** — 在地图上任意位置右键点击，显示经纬度和方圆 100 公里内的长征节点。

---

基于 Leaflet.js 构建，单页 HTML，打开即用。无需安装，无需联网（除地图瓦片）。
