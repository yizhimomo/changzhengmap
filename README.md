# 中国工农红军长征路线图

基于 Leaflet.js 构建的交互式长征路线可视化应用，单页 HTML 文件，开箱即用。

## 功能

- **交互地图** — 支持缩放、拖拽，覆盖 190+ 个标注节点，涵盖各支红军长征路线
- **时间轴滑块** — 按时间顺序浏览长征历程；选中节点时自动跳转到对应日期
- **搜索** — 按地名、标题、人物、日期搜索，结果按优先级排序
- **信息面板** — 每个节点包含详细描述、史料引用、参战部队及指挥官、会议记录、沙盘推演视频链接
- **图例切换** — 按方面军分组显示/隐藏路线，支持"显示全部/隐藏全部"
- **行军路线** — 各支红军以不同颜色路线绘制，含虚线段标注
- **右键菜单** — 显示经纬度（点击复制）、周边 100km 内节点列表（点击跳转）
- **键盘导航** — ← / ↑ 上一节点，→ / ↓ 下一节点（同方面军内）
- **日期显示** — 顶部实时显示当前时间轴日期

### 方面军路线

- **红一方面军**（中央红军）— 红色路线，江西→陕北
- **红二方面军**（红二、六军团）— 蓝色路线，湖南→甘肃
- **红四方面军** — 绿色路线，四川→甘肃
- **红二十五军** — 黄色路线，河南→陕北
- **北上抗日先遣队** — 紫色路线，闽浙赣→皖南

## 技术栈

- [Leaflet.js](https://leafletjs.com/) — 地图引擎
- 原生 JavaScript（无框架）
- 纯 CSS 主题
- 单文件 HTML（< 200 KB）

## 使用

浏览器直接打开 `可视化长征路线图.html` 即可，无需构建或服务器。

或使用 https://yizhimomo.github.io/changzhengmap/ 查看（清晰度有损）

## 关键节点

1. **1934年10月** — 江西瑞金出发
2. **1935年1月** — 遵义会议（确立毛泽东领导地位）
3. **1935年5月** — 强渡大渡河、飞夺泸定桥
4. **1935年6月** — 懋功会师（一、四方面军会合）
5. **1935年10月** — 到达陕北吴起镇
6. **1936年10月** — 三大主力会宁会师，长征胜利结束
